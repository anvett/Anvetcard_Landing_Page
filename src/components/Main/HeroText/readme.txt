# HeroText Component

## Descripción

El componente `HeroText` es la sección principal de la landing page, diseñada para captar la atención de los visitantes de inmediato. Presenta un título atractivo, una descripción convincente y un botón que, al hacer clic, lleva al usuario a la sección de precios de manera suave y animada.

## Características

- **Título y Subtítulo Dinámicos:** Presenta el nombre del producto y una breve descripción de su valor.
- **Botón de Acción:** Un botón que al hacer clic realiza un scroll suave hacia la sección de precios para facilitar la navegación del usuario.
- **Responsive:** Adecuado para todos los tamaños de pantalla, asegurando una experiencia de usuario coherente en dispositivos móviles, tabletas y escritorios.

## Uso

Para utilizar el componente `HeroText` en tu proyecto, simplemente impórtalo y úsalo dentro de tu aplicación React:

```jsx
import HeroText from './path/to/HeroText';

function App() {
  return (
    <div>
      <HeroText />
    </div>
  );
}
